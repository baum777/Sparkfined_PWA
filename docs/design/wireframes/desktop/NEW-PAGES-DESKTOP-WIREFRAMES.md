**Key Differences from Mobile:**
- **Filters:** Sidebar (left, sticky, 256px width) instead of full-width panel
- **Signal Cards:** 2-column grid instead of single column
- **Stats:** Horizontal 4-column (instead of 2x2)
- **Modal:** Centered with max-width-2xl (instead of slide-up)
- **Hover Effects:** More prominent (hover shows preview tooltip)

## Shared Desktop UX Patterns

### 1. Hover Effects
- **Cards:** Lift effect (`hover:-translate-y-2`) + border color change
- **Buttons:** Scale up slightly (`hover:scale-105`) + shadow glow increase
- **Links:** Text color change (zinc-400 â†’ zinc-100)
- **Tiles:** Background brightness increase

### 2. Sticky Elements
- **Navigation:** Fixed at top with backdrop blur
- **Sidebars:** Sticky positioning (top: 80px to clear nav)
- **Tooltips:** Appear on hover for additional context

### 3. Grid Layouts
- **2-Column:** Problem cards, access tiers
- **3-Column:** Feature cards, testimonials
- **4-Column:** Stats tiles
- **6-Column:** KPI tiles (BoardPage)
- **Custom:** BoardPage main grid (5fr-3fr-4fr)

### 4. Max-Width Constraints
- **Small Content:** max-w-2xl (672px) - HomePage
- **Medium Content:** max-w-4xl (896px) - Final CTA
- **Standard Content:** max-w-5xl (1024px) - Most sections
- **Wide Content:** max-w-6xl (1152px) - Feature grids
- **Full-Width:** max-w-7xl (1280px) - Hero, stats

### 5. Spacing
- **Section Padding:** py-20 â†’ py-32 (desktop)
- **Grid Gaps:** gap-4 â†’ gap-8 (desktop)
- **Card Padding:** p-4 â†’ p-6 (desktop)
- **Text Margins:** mb-6 â†’ mb-16 (desktop)

## Performance Optimizations

1. **Lazy Loading:** Images/sections load as user scrolls
2. **Code Splitting:** Route-level chunks (each page separate)
3. **CSS Animations:** GPU-accelerated (transform, opacity only)
4. **Debouncing:** Filter changes debounced by 300ms
