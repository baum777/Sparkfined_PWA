## Universal Desktop Enhancements

### 1. Hover States
All buttons gain hover effects:
- **Background:** `hover:bg-zinc-800`
- **Scale:** `hover:scale-105` (subtle)
- **Shadow:** `hover:shadow-lg` (on cards)
- **Cursor:** `cursor-pointer`

### 2. Modal Sizes
- **Mobile:** Full-screen overlay (100vw x 100vh)
- **Desktop:** Centered with max-width (max-w-4xl), backdrop blur

### 3. Sidebar Patterns (Future)
Not currently implemented, but could add:
- **Chart Tools:** Vertical sidebar (left/right)
- **Settings:** Sidebar nav + content area

### 4. Multi-Panel Layouts
- **Chart:** MiniMap + Timeline side-by-side
- **Journal:** Editor + Server Notes side-by-side (possible)
- **Notifications:** Server Rules + Trade Ideas side-by-side

### 5. Keyboard Shortcuts
More useful on desktop:
- **Global:** Cmd/Ctrl shortcuts work better
- **Chart:** Hotkeys for tools (H/T/F/Esc)
- **Focus States:** Visible ring on keyboard nav

### 6. Tooltip Enhancements
- **Desktop:** Hover tooltips on icons/buttons
- **Mobile:** Touch-hold tooltips (not impl.)
